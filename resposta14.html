A TAG para inserir uma fonte é: <font>
Ela é seguida de um atributo que altera as caracteristica do seu formato padrão,
( atributo="valor")


Para inserir em uma página um texto com fontes de tamanhos, cores e tipos diferentes utilizamos a TAG <font> ... <font>
Sintaxe do comando:
<font size=“n” face=“nome” color=“cor”>
