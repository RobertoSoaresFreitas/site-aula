Estilo Externos: Externo significa colocar as regras de CSS em um arquivo separado, e entao sua página HTML pode fazer um link para esse arquivo.
Essa abordagem permite definir regras em um ou mais arquivos que podem ser aplicadas em alguma página do seu web site.

Estilos Externos: Criar um arquivo com extensao css para utilizar no documento HTML, basta usar a tag no cabeçalho que referencia o arquivo css. Exemplo:
Arquivo fatec.css
H4 {
font-family : ‘Arial'
size:14pt;
Color: blue
}
